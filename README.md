#  HyunJaeGit 🐱
안정성과 유지보수성을 고려한 서버 사이드 시스템을 개발하고자 하는 신입 백엔드 개발자입니다. </br>
Legacy Spring Framework 기반 팀 개발 경험이 있으며, </br>
AWS 환경에서 보안 표준을 고려 서비스를 설계·배포한 경험이 있습니다.

## Technic 언어 및 기술 스택

[ Back-end ] ☕️
Java
JSP
Mybatis
Legacy Spring Framework
RESTful api
Spring Boot


[ Cloud ] ☁️
AWS (EC2, RDS, S3, VPC, IAM )

[ DataBase ] 💻
Oracle, MySQL

[ Front-end ] 🎨
HTML5, CSS3
Javascript

[ OS ] 💻
Linux (Oracle Linux7, CentOS, Ubuntu)
